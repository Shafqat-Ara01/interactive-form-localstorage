# Interactive Form with LocalStorage

This is a **mini frontend project** demonstrating how to create an interactive form that **persists user data** on the client-side using JavaScript's `localStorage`. The form also features a **dark/light mode toggle** and is fully **responsive**.

## Features

- Dark/Light mode toggle   
- Automatic saving of form inputs (text fields, radio buttons, checkboxes, textarea)  
- Restores saved input on page reload  
- Fully responsive design for desktop and mobile  
- Smooth user experience without losing data  

## Technologies Used

- HTML5  
- CSS3 (with responsive design)  
- JavaScript (ES6+)  
- Font Awesome for icons
